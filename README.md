mv-vim-nginx
============

Description
-----------

[Vim](http://www.vim.org/) plugin for [Nginx](http://www.nginx.org)

This plugin provides:

  * Syntax highlight
  * Snipmate snippets


Installation
------------

Use [Vundle](https://github.com/gmarik/vundle), to install this plugin:

    # inside vim
    :BundleInstall 'mv/mv-vim-nginx'

    # Add to ~/.vimrc
    Bundle 'mv/mv-vim-nginx'
